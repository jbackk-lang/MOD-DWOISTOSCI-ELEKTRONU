# Formalny stan elektronu

Elektron zapisany jako |O, S⟩.

Reguła energii:
E_O + E_S = const

Reguła wymiany:
dE_O = -dE_S

Oddziaływania:
- proton: O oddaje, S rośnie
- foton: O rośnie, S oddaje
- elektron: O oddaje, S rośnie (i odwrotnie)
- magnetron: E→O, B→S
- dziura: O oddaje, S rośnie → ekscyton

Model jest zgodny z TIMDR: wielowarstwowość, oscylacje, stabilizacja.
