# Magnetron jako test dwoistości elektronu

Magnetron = pole E + pole B.

Pole E pobudza warstwę O (orbitalną).  
Pole B pobudza warstwę S (spinową).

Pobudzenie jest przesunięte w fazie, więc:
- O rośnie, S maleje,
- potem S rośnie, O maleje.

Magnetron ujawnia pełną dwoistość elektronu: dwa sprzężone kanały energii krążące po dwóch torach.
