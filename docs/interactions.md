# Oddziaływania elektronu w modelu dwoistym

Elektron reaguje dwoiście, bo O i S są pobudzane inaczej.

Proton:
O oddaje energię → wiązanie
S rośnie → ustawienie spinu

Foton:
O rośnie → przejście orbitalne
S oddaje → flip spinu lub stabilizacja

Elektron:
O oddaje → odpychanie
S rośnie → spin alignment

Dziura:
O oddaje → pobudzenie zawisu
S rośnie → kompensacja
→ ekscyton

Magnetron:
E-pole → O
B-pole → S
→ pełna oscylacja dwoistości
